    • 3단계 (구조 개편): 서버 분리 및 AWS SQS 기반의 완전 비동기 아키텍처로 전면 개편. 서버 간 의존성을 제거하고, 유연한 확장성과 장애 격리가 가능한 MSA 기반 구조로 시스템 완성.
